### Array.prototype.map()

<code>map()</code> 方法创建一个新数组，其结果是该数组中的每个元素是调用一次提供的函数后的返回值。
#### 语法
```JS
const new_arr = arr.map(function callback(currentValue, index, array) {
  // 返回新数组
  return currentValue;
}, [, thisArg]);
```
#### 参数
- `callback`: 回调函数
- `currentValue`: 当前元素的值
- `index`: 当前元素的索引
- `array`: 原数组
- `thisArg`: 可选，作为回调函数的this对象
